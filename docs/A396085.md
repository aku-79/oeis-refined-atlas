# A396085

Publication track for the refined bracelet atlas class `(S1, sym)`.

Per OEIS review (July 2026) the DATA field is extended to 55 terms,
a(2)..a(56), via the closed formula

    a(n) = Sum_{d | n, d < n, d odd} g(d),

where `g(d)` counts aperiodic linear palindromes of odd length `d` over
partition-indexed content (the A395436 bisection): every counted word is a
power `U^(n/d)` of an aperiodic `(S1, ns)` word `U` of odd proper-divisor
length `d`.  In particular `a(p) = 1` for every prime `p`.  The Sawada
enumeration is kept as an independent check of the formula on small lengths.

The standalone OEIS program lives in `programs/a396085.py`.

The package module lives in `src/oeis_refined_atlas/a396085.py`.

The shared package combinatorics live in `src/oeis_refined_atlas/common.py`.
