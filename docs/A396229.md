# A396229

Publication track for the even-length refined bracelet atlas class
`(S2, ns)`. The OEIS index `n` represents underlying bracelet length
`2*n`; the sequence starts at offset 2.

The counted bracelets are reversal-closed, have exactly two distinct
linear-palindrome representatives, and have full rotational period.

Let `H(n)` count all `S2` cyclic orbits at underlying length `2*n`. An
even linear palindrome is determined by its first half, so the total number
of strongly normal linear palindromes of length `2*n` is `A005651(n)`. The
`S1` class at even length is entirely periodic and is counted by
`A396085(2*n)`. Since every `S2` orbit contains two linear-palindrome
representatives,

```text
H(n) = (A005651(n) - A396085(2*n)) / 2.
```

Möbius inversion over the rotational period extracts the full-period class:

```text
a(n) = Sum_{d | n} mu(n/d) * H(d)
     = (1/2) * Sum_{d | n} mu(n/d)
         * (A005651(d) - A396085(2*d)).
```

The current DATA range is `a(2)..a(20)`:

```text
1, 4, 22, 120, 795, 5729, 47728, 435456, 4439655, 49164467,
595788438, 7771511356, 109334263479, 1642874543650,
26350406591960, 448348912480608, 8080221295377684,
153591670339286219, 3073725730106911950
```

The standalone program evaluates the closed formula and contains Sawada's
fixed-content necklace algorithm as an independent direct check on small
indices.

The standalone OEIS program lives in `programs/a396229.py`.

The package module lives in `src/oeis_refined_atlas/a396229.py`.

The shared package combinatorics live in `src/oeis_refined_atlas/common.py`.

After installation, run:

```bash
oeis-a396229 20
```

or:

```bash
python -m oeis_refined_atlas.a396229 20
```
