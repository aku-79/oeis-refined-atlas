# A396051

Publication track for the even-length refined bracelet atlas class
`(S0, sym)`.  The OEIS index `n` represents underlying bracelet length
`2*n`; the sequence starts at offset 2 because the value at `n=1` is zero.

The counted bracelets are reversal-closed, have no linear-palindrome
representative, and have a nontrivial cyclic stabilizer.  Every such object
is a nontrivial power of a unique primitive `(S0, ns)` object counted by
A396632.  Consequently,

```text
a(n) = Sum_{d | n, d < n} A396632(d).
```

In particular, `a(p)=1` for every prime `p`.

The current DATA range is `a(2)..a(20)`:

```text
1, 1, 3, 1, 14, 1, 55, 12, 290, 1, 1824, 1, 12357, 299,
97740, 1, 863596, 1, 8428228
```

The standalone program evaluates the divisor formula and contains Sawada's
fixed-content necklace algorithm as an independent direct check on small
indices.

The standalone OEIS program lives in `programs/a396051.py`.

The package module lives in `src/oeis_refined_atlas/a396051.py`.

The shared package combinatorics live in `src/oeis_refined_atlas/common.py`.

After installation, run:

```bash
oeis-a396051 20
```

or:

```bash
python -m oeis_refined_atlas.a396051 20
```
