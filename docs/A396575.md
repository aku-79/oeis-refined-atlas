# A396575

Publication track for the cyclic multiplicity-collapse spectrum associated
with integer partitions.

For a partition `lambda` of `n`, let `c(lambda)` be the partition obtained by
listing, in weakly decreasing order, the multiplicities of the distinct part
sizes of `lambda`. Let

```text
C(n) = {c(lambda): lambda is a partition of n},
```

where each distinct collapse profile is included once.

For a partition `Q`, let `M(Q)` be the number of fixed-content necklaces with
color-multiplicity profile `Q`. Then

```text
a(n) = Sum_{Q in C(n)} M(Q).
```

For `Q=(q_1,...,q_k)` and `m=Sum_i q_i`, Burnside's lemma gives

```text
M(Q) = (1/m) * Sum_{d | gcd(q_1,...,q_k)}
       phi(d) * (m/d)! / Product_i (q_i/d)!.
```

The current DATA range is `a(1)..a(40)`:

```text
1, 2, 3, 5, 5, 12, 14, 26, 39, 68, 108, 220, 372, 677, 1191,
2368, 4167, 8125, 15163, 28453, 52865, 103208, 190573, 368201,
703725, 1362555, 2567667, 5056536, 9579999, 18464908, 36075194,
69357848, 132567002, 262039898, 499210661, 976415792, 1913373472,
3717786264, 7176336716, 14184357576
```

For example,

```text
C(4) = {(1), (1,1), (2), (2,1), (4)},
```

and each profile has one necklace, so `a(4)=5`.

At `n=6`, the ten profiles are `(1)`, `(1,1)`, `(2)`, `(1,1,1)`,
`(2,1)`, `(3)`, `(2,2)`, `(3,1)`, `(4,1)`, and `(6)`. Their necklace
counts are respectively `1,1,1,2,1,1,2,1,1,1`, giving `a(6)=12`.

The standalone program evaluates the Burnside formula and contains Sawada's
fixed-content necklace algorithm as an independent direct check through the
small reference range.

The standalone OEIS program lives in `programs/a396575.py`.

The package module lives in `src/oeis_refined_atlas/a396575.py`.

The shared package combinatorics live in `src/oeis_refined_atlas/common.py`.
