# A395436

Publication track for the refined bracelet atlas class `(S1, ns)`.

Per OEIS review (July 2026) the sequence is bisected: the class is empty in
even lengths, so `a(n)` is the count at length `2*n+1` with offset 1 (the
zeros at even lengths are omitted, as required by the OEIS style sheet).

The counted orbits are in bijection with aperiodic linear palindromes of odd
length over partition-indexed content, giving the closed formula

    a(n) = Sum_{d | 2*n+1} mu((2*n+1)/d) * T(d),

where `T(m)` counts all linear palindromes of odd length `m` (partitions of
`m` with exactly one odd part, central multinomial coefficient). The Sawada
enumeration is kept as an independent check of the formula on small lengths.

The standalone OEIS program lives in `programs/a395436.py`.

The package module lives in `src/oeis_refined_atlas/a395436.py`.

The shared package combinatorics live in `src/oeis_refined_atlas/common.py`.
